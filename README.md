 # Bujamart - Your Online Marketplace

**Bujamart** is an online eShop platform that allows users to buy and sell a wide variety of products, including **clothes**, **footwear**, **home appliances**, and much more. It provides a seamless shopping experience with the option to pay **on delivery** or via **PayPal**.

## Features

- **User-friendly interface**: Browse through various categories like **Men's Clothes**, **Women's Clothes**, **Footwear**, **Home Appliances**, **Watches**, and more.
- **Product Categories**:
    - **Men's Clothes**
    - **Women's Clothes**
    - **Footwear**
    - **Home Appliances**
    - **Lotion & Oils**
    - **Cooking Oil**
    - **Watches**
    - **Second-hand clothes** and more!
- **Flexible Payment Methods**: Choose between **Cash on Delivery** or **PayPal** for payment.
- **Responsive Design**: Fully responsive website that works seamlessly on desktop and mobile devices.
- **Multi-language Support**: Available in **English** and **French**.

## Technologies Used

- **HTML5**: For structuring the webpage.
- **CSS3**: For styling the website and creating a responsive layout.
- **JavaScript**: To make the website interactive, including adding products to the cart and processing checkout.
- **Firebase**: For authentication and data management (e.g., customer accounts, orders).
- **GitHub Pages**: To host the website.

## How to Run the Website Locally

To view the website on your local machine:

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/bujamart.git
    ```

2. Navigate to the project folder:
    ```bash
    cd bujamart
    ```

3. Open `index.html` in your preferred browser:
    ```bash
    open index.html
    ```
